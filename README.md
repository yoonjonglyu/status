# STATUS

**STATUS** is a human state modeling engine that quantifies real-world behavior into structured, dynamic, and decaying statistical vectors.

It is designed as the core layer for future AR/AI-integrated systems where human actions, decisions, and patterns are translated into measurable character-like attributes.

---

## Overview

STATUS is not a gamification layer.

It is a foundational statistical framework that:

- Converts real-world events into structured data
- Maps behavior to multidimensional state vectors
- Applies decay and temporal weighting
- Produces evolving human state snapshots
- Enables both personal tracking and multi-user comparison

This project represents the core logic engine for a future system where digital personas reflect measurable reality.

---

## Core Philosophy

1. **State, not score**  
   All values represent dynamic states, not permanent achievements.

2. **Decay is fundamental**  
   Every stat changes over time. Inactivity reduces state values.

3. **Events drive change**  
   The system is entirely event-based.

4. **Model versioning is mandatory**  
   Any modification to calculation logic must be version-controlled.

5. **Separation of layers**  
   - Human-readable philosophy and definitions → Markdown
   - Machine-readable execution logic → JSON
